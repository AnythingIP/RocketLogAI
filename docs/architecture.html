RocketLogAI v2 Architecture

Local-first security platform — four products, one unified AI brain.

🚀 RocketLogAI
Syslog, threats, AI assistant, devices, briefing
🔧 RocketRemediate
Dry-run, approval, backup, rollback
🛡️ RocketShield
WAF, AV, parental controls (inline/SPAN)
📱 RocketAI Mobile
QR pairing, sync, remote control API

Data Flow

Syslog UDP/TCP (5140) ──► Parser ──► SQLite Storage
                              │
                              ├──► Rule Engine ──► Threats
                              ├──► UEBA Detector ──► Anomalies
                              └──► Vector Store (RAG) ──► Unified AI Brain
                                        │
                    ┌───────────────────┼───────────────────┐
                    ▼                   ▼                   ▼
              AI Assistant        Daily Briefing      MCP Server
              RocketRemediate     Mobile API          External AI Clients

Core Modules

  • logsentinel/brain/ — VectorStore, RAG, ConversationMemory, AIOrchestrator
  • logsentinel/mcp/ — MCP server (stdio). Run: logsentinel mcp
  • logsentinel/remediate/ — Approval workflow, backups, rollback plugins
  • logsentinel/shield/ — WAF, AV scanner, parental controls
  • logsentinel/ueba/ — Behavioral anomaly detection + explainable reports
  • logsentinel/audit/ — Tamper-evident audit trail
