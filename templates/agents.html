Manual install (after register)

        
Save the install_token from registration (shown once). Then on the target machine:

        
# Linux/macOS — replace SERVER and paths from registration response
curl -fsS SERVER/api/v2/agents/AGENT_ID/install.sh -o install-agent.sh
chmod +x install-agent.sh
# Edit script with your token, then run ./install-agent.sh

# Windows PowerShell
Invoke-WebRequest SERVER/api/v2/agents/AGENT_ID/install.ps1 -OutFile install-agent.ps1
