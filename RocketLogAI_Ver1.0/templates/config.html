Configuration

Manage all major RocketLogAI settings from the browser. Changes are written directly to your config.yaml.

Quick Guidance
  • LLM section: Point at your local model server or a cloud provider. Test connection before saving.
  • Web / SSL: Both HTTP and HTTPS are on by default. Self-signed certs are generated automatically for easy HTTPS.
  • Geo: Enable offline IP geolocation. Provide path to GeoLite2-City.mmdb or leave blank for auto-detect.
  • Blacklist: External IPs on public blocklists get automatically escalated to high risk.
  • Devices + AI: RocketLogAI learns normal behavior per device (including MAC history) and uses AI to decide trust vs. investigate. It can now also identify the manufacturer from the MAC address.
